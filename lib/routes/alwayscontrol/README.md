# Always Control

## 最新动态

### 路由

`/alwayscontrol/news`

### 参数

无

### 示例

`https://rsshub.app/alwayscontrol/news`

### 说明

Always Control（旭衡电子）智能能源管理系统解决方案专家的最新动态。

该路由会获取官网新闻列表页的所有文章，并抓取每篇文章的完整内容，包括：
- 文章标题
- 发布日期
- 完整正文内容
- 文章配图
- 作者信息
- 分类标签

### 数据源

- 新闻列表：https://www.alwayscontrol.com.cn/zh-CN/news/list
- 新闻详情：https://www.alwayscontrol.com.cn/zh-CN/news/detail
