# RSSHub

> 🍭 使用 RSS 连接全世界

## 介绍

RSSHub 是一个轻量、易于扩展的 RSS 生成器，可以给任何奇奇怪怪的内容生成 RSS 订阅源

[使用文档](https://rsshub.js.org)

当前支持列表：

- bilibili
  - 番剧
  - UP 主投稿
  - UP 主动态
  - UP 主收藏夹
  - 分区视频
- 微博
  - 博主
- 网易云音乐
  - 歌单
- 掘金
  - 分类
- 简书
  - 首页
  - 7日热门
  - 30日热门
  - 专题
  - 作者
- 知乎
  - 收藏夹
- 自如
  - 房源
- 快递
